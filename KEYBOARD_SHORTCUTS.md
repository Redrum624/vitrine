# Keyboard Shortcuts Reference

Quick reference guide for all keyboard shortcuts in Vitrine.

---

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+O` | Open Image | Open file selection dialog to load an image |
| `Ctrl+S` | Save | Save current image (if path exists) |
| `Ctrl+E` | Export | Open export dialog with format options |

---

## Edit Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo last change (50-state history) |
| `Ctrl+Shift+Z` or `Ctrl+Y` | Redo | Redo previously undone change |
| `Ctrl+R` | Reset All | Reset all adjustments to default values |

---

## View & Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl++` or `Ctrl+=` | Zoom In | Increase canvas zoom level |
| `Ctrl+-` | Zoom Out | Decrease canvas zoom level |
| `Ctrl+1` | Actual Size | Set zoom to 100% (1:1 pixel mapping) |
| `Ctrl+0` | Fit to Window | Fit image to window size |

---

## Display Options

| Shortcut | Action | Description |
|----------|--------|-------------|
| `V` | Toggle Thumbnail Panel | Show/hide thumbnail image browser |
| `M` | Toggle Module Panel | Show/hide adjustment modules panel |
| `C` | Toggle Crop Overlay | Show/hide crop grid and handles |
| `B` | Toggle Before/After | Compare original vs edited image |

---

## Rating

| Shortcut | Action | Description |
|----------|--------|-------------|
| `1`–`5` | Rate Image | Set a 1–5 star rating on the current image (written to the file as `xmp:Rating`) |
| `0` | Clear Rating | Remove the rating from the current image |

You can also rate from the star controls on each filmstrip thumbnail or the larger
star overlay at the bottom-right of the canvas.

---

## Processing

| Shortcut | Action | Description |
|----------|--------|-------------|
| `P` | Apply Preset | Apply selected preset to current image |
| `B` | Batch Process | Open batch processing dialog |
| `M` | Toggle Mask Overlay | Show/hide adjustment masks |

---

## Help & Tools

| Shortcut | Action | Description |
|----------|--------|-------------|
| `F1` or `?` | Keyboard Shortcuts | Show this shortcuts help dialog |

---

## Tips

### Shortcut Conflicts
- Shortcuts are disabled when typing in text fields
- Dialog-specific shortcuts only work when dialog is open
- Global shortcuts work from anywhere in the application

### Custom Shortcuts
- Currently, custom shortcuts are not supported
- Planned for future release

### Module-Specific Shortcuts
- Some modules have additional shortcuts when focused
- Check module documentation for details

---

## Export Shortcuts (in Export Dialog)

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Enter` | Export | Confirm and start export process |
| `Esc` | Cancel | Close export dialog without exporting |
| `Tab` | Navigate | Move between export settings |

---

## Crop & Transform Shortcuts (when active)

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Enter` | Apply | Apply crop/transform changes |
| `Esc` | Cancel | Cancel and revert changes |
| `R` | Reset | Reset to full image dimensions |
| `H` | Flip Horizontal | Mirror image horizontally |
| `V` | Flip Vertical | Mirror image vertically |
| `A` | Auto-Straighten | Detect and fix horizon tilt |

---

## Workflow Tips

### Fast Editing Workflow
1. `Ctrl+O` - Open image
2. Make adjustments using mouse/sliders
3. `Ctrl+E` - Export with settings
4. `Enter` - Confirm export

### Comparison Workflow
1. Edit your image
2. `B` - Toggle before/after
3. `Ctrl+Z` - Undo if needed
4. `Ctrl+Shift+Z` - Redo to compare

### Batch Processing Workflow
1. `Ctrl+B` - Open batch dialog
2. Select images
3. Choose output settings
4. Let it process

## Platform-Specific Notes

### Windows
- `Ctrl` is the primary modifier key
- `Alt` for alternative actions

### macOS
- `Cmd` replaces `Ctrl` in most shortcuts
- `Option` replaces `Alt`

### Linux
- Same as Windows (Ctrl-based)

---

## Accessibility

### Screen Reader Support
- All shortcuts announced by screen readers
- Focus indicators visible on all controls
- Keyboard navigation supported throughout

### High Contrast Mode
- Shortcuts work in all display modes
- Visual indicators for active shortcuts
