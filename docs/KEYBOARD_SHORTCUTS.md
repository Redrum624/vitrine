# Keyboard Shortcuts Reference

## ⌨️ **Professional Photo Editing Shortcuts**

Vitrine provides comprehensive keyboard shortcuts for efficient photo editing workflows. Master these shortcuts to dramatically speed up your editing process.

## 📁 **File Operations**

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+O` | Open Image | Open file browser to load an image |
| `Ctrl+S` | Save/Export | Open export dialog to save your work |
| `Ctrl+Shift+S` | Export As | Export with different settings |
| `Ctrl+N` | New Project | Start a new editing session |
| `Ctrl+W` | Close Image | Close current image |
| `Ctrl+Q` | Quit Application | Exit Vitrine |

## 🎯 **Pro Tips for Efficiency**

### **Workflow Shortcuts**
1. **RAW Processing**: `Ctrl+O` → `Shift+A` → Fine-tune → `Ctrl+S`
2. **Quick Exposure Fix**: `2` → `A` → Adjust exposure slider → `R` if needed
3. **Color Correction**: `3` → `W` → Click neutral area → Fine-tune
4. **Batch Similar Images**: `Ctrl+Shift+C` → Next image → `Ctrl+Shift+V`

### **Speed Editing Techniques**
- **Use number keys** to quickly jump between modules
- **Hold Shift** while adjusting sliders for fine control
- **Double-click sliders** to reset individual parameters
- **Use Tab** to cycle through modules during review
- **Press Space** frequently to see overall composition

### **Advanced Combinations**
- `Ctrl+Shift+A` - Auto-adjust all modules at once
- `Ctrl+Alt+R` - Reset and auto-adjust current module
- `Ctrl+B` - Quick before/after comparison
- `Shift+Click` on auto buttons - Apply to similar parameters

## 💡 **Memory Aids**

### **Common Patterns**
- **Numbers (1-8)**: Module selection
- **Letters**: Tools (W=White Balance, C=Crop, etc.)
- **Ctrl+Letter**: File operations
- **Function Keys**: Panel toggles
- **Shift+Action**: Enhanced version of action

### **Mnemonic Devices**
- **R** = Reset (remember "R for Reset")
- **A** = Auto-adjust (remember "A for Auto")
- **B** = Before/after (remember "B for Before")
- **G** = Grid/guides (remember "G for Grid")
- **H** = Histogram (remember "H for Histogram")

## 🎓 **Learning Path**

### **Beginner (Master these first)**
1. `Ctrl+O` - Open images
2. `Space` - Fit to screen
3. `1-8` - Module navigation
4. `A` - Auto-adjust
5. `Ctrl+S` - Export

### **Intermediate**
1. `R`/`Shift+R` - Reset modules
2. `Ctrl+Z`/`Ctrl+Y` - Undo/redo
3. `B` - Before/after comparison
4. `Ctrl++`/`Ctrl+-` - Zoom control
5. Tool shortcuts (`W`, `C`, `S`)

### **Advanced**
1. `Ctrl+Shift+C`/`V` - Copy/paste all settings
2. Panel toggles (`F1`-`F4`)
3. Performance shortcuts (`Ctrl+Shift+P`)
4. Batch operations (`Ctrl+Shift+B`)
5. Custom shortcuts and workflows
